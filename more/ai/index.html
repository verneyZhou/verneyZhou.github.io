# AI工具

  • cursor (opens new window): 官网标题是：The AI-first Code Editor, 感兴趣的可以试一下

  • 博特妙笔 (opens new window): 公文写作AI工具

  • Dify (opens new window): Dify 是一款开源的大语言模型(LLM) 应用开发平台。它融合了后端即服务（Backend as Service）和 LLMOps 的理念，使开发者可以快速搭建生产级的生成式 AI 应用。

  • Sincode.AI (opens new window): AI工具，可聊天、生成图片、创建小工具等

  • 通义灵码 (opens new window): 阿里出的AI编码助手, 类似于 github 的Copilot

  • Gliglish (opens new window): 免费的AI外语语音聊天工具，可支持不同国家语言

  • lalal.AI (opens new window): 从任何音频和视频中提取人声、伴奏和各种乐器

  • Idify (opens new window): 制作证件照

  • boardmix (opens new window): 博思白板，一个点燃团队协作和激发创意的空间，集AIGC，一键PPT，思维导图，笔记文档多种创意表达能力于一体，将团队工作效率提升到新的层次。

  • Docus AI (opens new window): 与AI健康助手交谈，生成您的健康报告，并与来自美国和欧洲的顶级医生进行验证。

# AI项目收藏

  • AutoGPT (opens new window): 利用ChatGPT自动生成工程项目；github上目前有150+k个⭐️了，算是AI的明星项目了

  • screenshot-to-code (opens new window): 基于截图页面生成前端项目，使用gpt-4-vision-preview识别图片并配合提示词，直接让GPT4生成相应的HTML代码；参考 (opens new window)

  • highcharts (opens new window): 通过ChatGPT生成图表

  • draw-a-ui (opens new window): 通过画图生成网页代码

  • DB-GPT (opens new window): DB-GPT是一个开源的数据库领域大模型框架。目的是构建大模型领域的基础设施，通过开发多模型管理、Text2SQL效果优化、RAG框架以及优化、Multi-Agents框架协作等多种技术能力，让围绕数据库构建大模型应用更简单，更方便。DB-GPT: 一个隐私的数据库垂直领域本地化大模型框架 (opens new window)

  • GeminiProChat (opens new window): 基于 Gemini 搭建的Chat项目；媲美GPT4，一键搭建谷歌 Gemini (opens new window)
