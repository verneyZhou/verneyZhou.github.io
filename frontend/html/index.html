    1.mpvue项目启动时，通过微信开发者工具打开项目报错： Q:

    A:app.json是小程序的入口配置文件，项目中丢失可能打包过程中未生成该文件。 Q:微信开发者工具添加mpvue项目时报错：

    A:mpvue项目通过npm run dev 命令启动，并将项目打包到dist文件夹中，再通过微信开发者工具打开项目dist目录，可运行小程序；报错是因为打包没有生成小程序的app.json文件，打开项目package.json，作如下修改： "mpvue-loader": "^1.0.13", => "mpvue-loader": "1.0.13", 再删除node包，npm install , npm run dev 重新打包生成dist文件即可。 https://segmentfault.com/a/1190000015812634?utm_source=tag-newest http://blog.sina.com.cn/s/blog_60cf05130102xz1v.html 注：‘1.0.13’ 表示安装指定的版本号，也就是安装1.0.13版本。 ‘~1.0.13’ 表示安装1.0.x的最新版本。 ‘^1.0.13’ 表示安装1.x.x的最新版本。
