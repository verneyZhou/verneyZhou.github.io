              Vue第三方组件插件库收集

              # Vue第三方组件插件库收集

              # UI库

              • ElementUI (opens new window)

              一套为开发者、设计师和产品经理准备的基于 Vue 的桌面端组件库

              • iView (opens new window)

              基于 Vue.js 3 的企业级 UI 组件库和前端解决方案。

              • Vuetify (opens new window)

              Vuetify 是一个纯手工精心打造的 Material 样式的 Vue UI 组件库。

              • Vant (opens new window)

              有赞出的移动端 Vue 组件库

              • Naive UI (opens new window)

              Naive UI 是一款基于当前比较新的 Vue 3.0/TypeScript 技栈开发的前端 UI 组件库。github地址 (opens new window)

              # 工具类

              • Vue Apollo (opens new window)

              在你的 Vue.js 应用中集成 GraphQL

              • VueUse (opens new window)

              vue工具库

              5 个可以加速开发的 VueUse 函数库 (opens new window)

              • FormMaking (opens new window)

              基于 Vue 的可视化低代码表单设计器，赋能企业实现低代码开发模式；帮助开发者从传统枯燥的表单代码中解放出来，更多关注业务，快速提高效率，节省研发成本。github地址 (opens new window)

              • Pinia (opens new window)

              Pinia 是最新一代的 Vue 轻量级状态管理库。它适用于 Vue 2.x 和 Vue 3.x, 类似 Vuex

              • Vue-Chartjs (opens new window)

              vue-chartjs 是一个 Vue 对于 Chart.js 的封装，让用户可以在Vue中轻松使用Chart.js，很简单的创建可复用的图表组件，非常适合需要简单的图表并尽可能快地运行的人。vue-chartjs抽象了基本逻辑，同时也暴露了Chart.js对象，让用户获得最大的灵活性。它支持 Vue 3 和 Vue 2。github地址 (opens new window)

              • Vue-ECharts (opens new window)

              Vue-ECharts 是 Apache ECharts (opens new window) 的 Vue.js 组件。使用 Apache ECharts 5，同时支持 Vue.js 2/3。github地址 (opens new window)

              • Trois (opens new window)

              Trois 是一个基于 Three.JS 的 Vue 3 可视化库，ThreeJS + VueJS 3 + ViteJS, 它是一个流行的 WebGL 库。Three.JS 对桌面和移动设备都有很好的支持。该库允许我们使用 VueJS 组件轻松为网站创建 3D 效果。github地址 (opens new window)

              • Popmotion (opens new window)

              Popmotion 是一个只有12KB的 JavaScript 运动引擎，可以用来实现动画，物理效果和输入跟踪。原生的DOM支持：CSS，SVG，SVG路径和DOM属性的支持，开箱即用。

              # 插件库

              • Vue-Draggble (opens new window)

              Vue Draggable 是一个基于 Sortable.js 的 Vue 拖拽组件。github地址 (opens new window)

              • 虚拟滚动：vue-virsual-scroller

              • 图片资源懒加载：vue-lazyload

              # 组件库

              • vue-simple-spinner

              一个基于Vue.js的简单、弹性的加载样式旋转器。参考 (opens new window)

              • vue-radial-progress

              这是一个径向进度条效果的加载器组件，使用svg和javascript绘制带有渐变径向进度条效果的加载器，可以用作加载、进度提示。参考 (opens new window)

              • vue-loading-overlay

              这是一个基于Vue.js的全屏加载组件，既可以用作组件(<loading></loading)，又可以按插件(Vue.$loading())的方式调用。参考 (opens new window)

              • vue-progress-path

              这是一个可以设置任何路径的进度条和加载器。默认是圆形路径，内置半圆、线条、方形路径，你甚至可以传入自定义svg路径。参考 (opens new window)

              # 其他

              • VuePress (opens new window)

              VuePress 是一个基于 Vue 的轻量级静态网站生成器，以及为编写技术文档而优化的默认主题。它是为了满足 Vue 自己的子项目文档的需求而创建的。

              • VitePress (opens new window)

              VitePress 是 VuePress 的继承者，建立在vite 之上。github地址 (opens new window)
